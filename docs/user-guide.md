## 左侧：会话列表

### 分组
- 按项目目录 (`projectDir`) 自动分组
- 每组显示文件夹名 + 会话数量
- 点击文件夹头折叠/展开
- 无项目目录的会话归入"无项目目录"

### 搜索
- 实时过滤，匹配：标题、摘要、项目目录名、sessionId
- 按 Esc 清空搜索

### 批量管理
- 点击 ☑ 进入批量模式
- 勾选会话 → 批量删除
- 支持按文件夹全选/取消
- 再次点击 ☑ 退出

## 右侧：会话详情

### 头部
- 会话标题
- 项目路径（点击复制）
- 最后活跃时间
- resume 命令（可复制）
- ▶ 恢复按钮：在终端中启动 `claude --resume`
- 🗑 删除按钮

### 消息列表
- 消息按时间正序显示（最早在上，最新在下）
- 首次加载自动滚动到底部
- 向上滚动到顶部自动加载更早消息
- 显示加载进度 "50 / 531"

### 消息样式
| 角色 | 颜色 | 位置 | 标签 |
|------|------|------|------|
| user | 绿色 | 左缩进 | You |
| assistant | 蓝色 | 右缩进 | AI |
| tool | 紫色 | 居中 | Tool |
| system | 琥珀色 | 居中 | System |

### Markdown 渲染
- 使用 marked.js 渲染消息内容为 HTML
- 支持代码块、表格、列表、引用、标题
- 代码块深色背景 + 等宽字体，横向滚动

### 思考过程
- assistant 消息中的 thinking 块渲染为引用样式
- 格式：`> 🤔 **思考过程**` + 缩进文本

### 工具调用
- 显示工具名：`**调用工具: Bash**`
- 显示参数 JSON（代码块格式）
- 工具输出：`**输出:**` + 内容代码块
- 错误输出：`**输出 (错误):**` 标记

### 消息折叠
- 超过 3000 字符的消息自动折叠
- 点击"展开完整内容"查看全部
- 折叠渐变遮罩适配消息角色颜色

## 键盘快捷键

| 按键 | 功能 |
|------|------|
| j | 下一个会话 |
| k | 上一个会话 |

## 侧边栏

- 右侧边缘可拖拽调整宽度（220px ~ 600px）

## 暗色模式

- 自动跟随系统 `prefers-color-scheme`
- CSS 变量定义全部颜色，无缝切换
